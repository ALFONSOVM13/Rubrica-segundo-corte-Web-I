# Rubrica-segundo-corte-Web-I
Diseñe un sitio web totalmente responsive cuyo contenido principal será cargado desde una API.
incluya header,la información principal del sitio está constituida por 12 cards organizadas de la siguiente manera:
     1.Para la resolución mas grande de pantalla: 4 cards en cada fila. El sitio contará con 3 filas para un total de 12 cards. 
     2.Para la resolución mas pequeña de pantalla: se deben mostrar las 12 cards una a una en una sola columna
     3. Incluya un footer con información del desarrollador.
     4. Cada una de las cards debe incluir imagen y titulo cargado de la api:
      https://jsonplaceholder.typicode.com/photos
    5. Puede usar las siguientes API opcional, en ese caso la card debe incluir imagen y nombre del personaje:
      https://rickandmortyapi.com/api/character
    6. Debe entregar un archivo de texto con links del repositorio en linea y despliegue.
    7. Incluya una sección de filtro tipo lista desplegable (select) donde se carguen los titulos/nombres de las cards como opciones.  al seleccionar una de ellas se de     mostrar solo esa card. una de las opciones debe permitir ver todas las 12 cards.
